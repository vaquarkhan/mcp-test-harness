# Plugin and integration registry (proposed)

MCP Test Harness already supports extension points (assertions, fixtures, reporters, transports).  
This page is the placeholder for a curated registry of official/community plugins.

## Registry goals

- discover integrations quickly
- avoid duplicate one-off plugins
- document support level and maintenance status

## Suggested categories

- CI/reporting integrations (GitHub Checks, Slack/Teams/Discord, etc.)
- Security scanners
- Governance/policy adapters
- Data providers and test case loaders
- Observability exporters

## Listing template

- Name
- Category
- What it adds
- Install path
- Configuration example
- Maintenance status
