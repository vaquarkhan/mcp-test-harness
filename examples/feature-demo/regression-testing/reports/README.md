# Regression demo reports

These files show where report artifacts are generated for the regression demo pack.

## Generate reports

```bash
mcp-test -c examples/feature-demo/regression-testing/mcp_test_regression_demo.yaml
```

## Artifacts

- `regression-demo.html` - dashboard-style HTML report
- `regression-demo.json` - machine-readable summary and metadata
- `regression-demo.xml` - JUnit XML for CI systems
