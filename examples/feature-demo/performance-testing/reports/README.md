# Performance demo reports

These files show where report artifacts are generated for the performance demo pack.

## Generate reports

```bash
mcp-test -c examples/feature-demo/performance-testing/mcp_test_performance_demo.yaml
```

## Artifacts

- `performance-demo.html` - dashboard-style HTML report
- `performance-demo.json` - machine-readable summary and metadata
- `performance-demo.xml` - JUnit XML for CI systems
