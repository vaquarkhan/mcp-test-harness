# Functional demo reports

These files show where report artifacts are generated for the functional demo pack.

## Generate reports

```bash
mcp-test -c examples/feature-demo/functional-testing/mcp_test_functional_demo.yaml
```

## Artifacts

- `functional-demo.html` - dashboard-style HTML report
- `functional-demo.json` - machine-readable summary and metadata
- `functional-demo.xml` - JUnit XML for CI systems
